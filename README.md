## 🚀 Deployment Guide (Vercel)

1. Go to [Vercel](https://vercel.com/)
2. Click on “New Project”
3. Import your GitHub repo `spotify-clone`
4. Keep all settings default (no frameworks)
5. Hit Deploy 🚀
6. Boom. Done.
